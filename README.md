
# Tours and Travel Website
- Embedds the Home Page
- Responsive Website
- Includes
    - Booking Page
    - Best Selling Packages
    - Popular Destinations
    - Explore by themes
    - Newsletter
    - Benefits of using this travel website
    - Reviews
    - FAQs
    - Membership plans







## Tech Stack

**Client:** HTML, CSS, JS
